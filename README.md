# Projects_management_2

### by Daniel Mestawat
### Description
This Application can be used by clonning it, open in command prompt or bash, or use the link mentioned below.The purpose of this application is to create a project, developers and assigne a developers needed for a particulart project where needed. This application is a full MERN application where it uses MongoDB as a backend database,Expreess as a modular web framework for Node,React as a front-End frame work, and node.js as web servers.



### HOW TO USE THE APPLICATION

- Clone this repository.
- Open in command prompt(cmd) or bash.
- Initialize by typing npm run i.
- start the react app by typing npm run start.



### Technologies Used

- Node.js
- mongo databasee
- mongoose
- express
- robo 3T
- morgan
- react as a front-end frame work.
- Axios
- body-parser
- cors
